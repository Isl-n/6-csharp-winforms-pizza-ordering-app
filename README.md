# C# WinForms Pizza Ordering App

A Windows Forms application that simulates a simple pizza ordering system.
The application allows users to customize a pizza and see the total price update immediately based on their selections.

This project was built to practice **WinForms UI development**, **event handling**, and **real-time UI updates**.
